# Usage / Installation  
`Require` Screen capture (GIF)  

```bash  
git clone git@github.com:7rs/pages.git  
# HTTPS: git clone https://github.com/7rs/pages.git  
# "--depth 1" option for faster speed  

pnpm install astro  
pnpm run dev  
```
