## Installation  

  ```bash
  git clone git@github.com:7rs/pages.git --depth 1
  # git clone https://github.com/7rs/pages.git --depth 1

  cd pages

  pnpm install astro
  # npm install astro
  # yarn add astro

  pnpm run dev
  # npm run dev
  # yarn run dev
  ```
